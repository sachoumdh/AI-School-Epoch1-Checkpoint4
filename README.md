# "VoiceBread" Multimedia Systems
### 1. Project Description 
This project explores multimodal AI by combining image and audio processing technologies, the system integrates:
* Optical Character Recognition (OCR) to extract text from images
* Automatic Speech Recognition (ASR) to convert speech from audio into text
By the end of this project, the system is able to:
* Read text from an image using OCR
* Transcribe speech from audio using ASR
* Optionally, combine both outputs into a single text document
* Optionally, summarize the generated notes and produce an audio summary

This project was implemented as part of AI School – Epoch 1 – Checkpoint 1.4
